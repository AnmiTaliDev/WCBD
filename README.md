# WCBD — What Can Be Done?

A curated list of project ideas across different categories — for those who have plenty of ideas but nowhere to write them down.

> I often get ideas when I'm away from my computer, but when I sit down at it, I forget them and end up with nothing to do. So I decided to start writing them down — for myself and for anyone else who's bored and looking for something to build.
