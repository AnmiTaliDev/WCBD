# Contributing to WCBD

Thank you for wanting to contribute! This is a shared list of project ideas — no code required.

## How to Add an Idea

1. Fork this repository.
2. Edit `README.md` and add your idea to the appropriate category table.
3. If no category fits, you can propose a new one.
4. Open a Pull Request with a short description of your idea.

## Format

Each idea follows this format in the table:

| Idea | Description | Example Implementation |
|------|-------------|------------------------|
| Short name | What it is and why it's interesting | `author/repo` or `—` if none |

- **Idea**: a short, clear name.
- **Description**: one or two sentences explaining what should be built.
- **Examples**: one or more existing projects that implement or are close to the idea. Separate multiple examples with `<br>`. If there are no examples yet, use an issue link: `[Add an example](https://github.com/AnmiTaliDev/WCBD/issues/new?title=Example+for%3A+<idea name>)`.

## Rules

- Ideas only — no code in this repository.
- Keep descriptions concise and in English.
- One idea per Pull Request (preferred).
- Be respectful and constructive.

## Adding Yourself as a Contributor

After your PR is merged, add your name to [CONTRIBUTORS.md](CONTRIBUTORS.md).
